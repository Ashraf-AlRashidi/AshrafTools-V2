## 🌟 What is AshrafTools?

> A **full-featured learning platform** packed into a single HTML file — built for Software Engineering students to practice Java interactively, compete on leaderboards, and get instant AI feedback on their code.

<br/>

<div align="center">

| 🔥 | Stat | Value |
|:---:|:---|:---:|
| 🧩 | Total Challenges | **30 Problems** |
| 🏆 | Max Points | **~1,500 pts** |
| 👥 | Active Students | **200+** |
| 🤖 | AI Judgements | **1,000+** |
| ⭐ | GitHub Stars | **128** |

</div>

<div align="center">

|:---|:---|
| 🧩 **Java Challenges** | 30 problems across 4 difficulty levels |
| 🤖 **AI Judge** | ChatGPT reviews your code & gives Arabic feedback instantly |

## 🤖 How AI Judge Works

```
Student writes code  →  ChatGPT reads it  →  Verdict  →  Instant Feedback
        ↓                      ↓               ↓                ↓
   Java Code             GPT-4o-mini       ✅ Correct       Arabic explanation
                                           ⚠️ Partial       + Points awarded
                                           ❌ Wrong         Hint to improve
```
